CrUD для таблицы внутри файла.
Программа запускается с одним из следующих наборов параметров:
c productName price quantity
u id productName price quantity
d id

"c" - добавление товар с заданными параметрами в конец файла.
"u" - обновление данных товара с заданным id.
"d" - удаление товара с заданным id.

id -id товара,  8 символов.
productName - название товара, 30 символов.
price - цена товара 8 символов.
quantity - количество товара, 4 символа.
